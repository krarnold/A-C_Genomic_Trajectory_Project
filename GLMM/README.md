## Old vs New Population Comparisons
Scripts for comparing founder populations to reciprocal trajectory 
populations using permutation-based GLM analysis.
- GLMM_AC_OldvNew_Observed.R: Quasibinomial GLM comparing old and new populations
- GLMM_OldvNew_Permutation.R: Permutation-based null distribution for old vs new comparisons
- GLMM_AC_OldvNew_Perm_Plots.R: FDR significance and plotting for old vs new comparisons
